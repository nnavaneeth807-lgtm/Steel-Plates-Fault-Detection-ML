# Steel Plates Fault Detection using Machine Learning

## Project Overview
This project uses Machine Learning techniques to detect faults in steel plates. The Steel Plates Faults dataset was used to train and evaluate different classification models.

## Objectives
- Analyze steel plate fault data
- Build classification models
- Compare model performance
- Identify the best algorithm for fault detection

## Dataset
Steel Plates Faults Dataset from the UCI Machine Learning Repository.

## Algorithms Used
1. Random Forest
2. Decision Tree
3. K-Nearest Neighbors (KNN)

## Tools and Libraries
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## Results
Among the implemented models, Random Forest achieved the best overall performance for steel plate fault classification.
